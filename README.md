## Get it running
To train the models:
1. Make sure python, numpy, pandas, scikit-learn, tensorflow, and matplotlib are available locally. You can use a Python virtual environment if you want.
2. Use `python3 NeuralNet.py` or click the Run button on the NeuralNet.py file in your IDE to run the program.
3. That's it!
